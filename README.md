# Calorie Bear

Calorie and nutrition tracker for iOS.

Log meals by photo or search, set goals based on your body stats, and track protein, carbs, and fat — all in one place.

## Stack

- **Mobile** — React Native (Expo SDK 54)
- **Backend** — Node.js + Express, deployed on Railway
- **Database** — MongoDB Atlas
- **AI** — Claude Vision API for food photo analysis
